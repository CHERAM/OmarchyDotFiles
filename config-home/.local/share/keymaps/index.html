Tips & Milestones
5 Rules for learning Vim:
1. Stop using arrow keys — use hjkl only.
2. One new thing per day — learn it, use it 20× until it's muscle memory.
3. Never reach for the mouse — if you want the mouse, find the Vim way instead.
4. Use . (dot) constantly — if you do something twice, the third time use dot repeat.
5. Motions > line numbers — think in words, objects, and targets, not row/col coords.
Week 1
[ ] Can open, edit, save, and quit without touching the mouse
Week 2
[ ] Navigate any file using w/b/f/t and text objects instead of arrow keys
Week 3
[ ] Full JS/TS edit loop: gd, gr, K, ca, cf — no VS Code needed
Week 4
[ ] Use macros and * → cgn → . for bulk edits
Week 5+
[ ] Faster in Vim than VS Code for daily JS/TS tasks
